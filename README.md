#Building and Running

	make -j 5
	./shell

#Features
- All basic commands
- semicolon seperated multiple commands
- Piping implemented
- Foreground and Background
- List running Jobs
- Killing all background jobs
- exit and quit commands for exiting the shell
- Signal handling like Ctrl+D, Ctrl+Z and Ctrl+C
- Single Header File

#Tested On
- gcc version 5.3.1
- Fedora 23
